Django Tiny Shop
==========================

Super simple and small Django Cart/shop. 


TRY
=============

1. Create a virtualenv
2. Install requirements.txt
	pip install -r requirements.txt
3. Sync your db
	./manage.py syncdb
	
4. launch demo site:
	./manage.py runserver
	
5. Browse your shop
	http://localhost:8000/admin
